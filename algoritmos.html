Metodo de Ordenação

BubbleSort

O Bubble sort é um algoritmo simples de ordenação que recebe como entrada uma lista de elementos e produz uma lista ordenada de acordo com um critério.

Ordenação do seguinte array: [13, 18, 25, 12, 1, 5]



let res = document.getElementById('res')
            let array = [13, 18, 25, 12, 1, 5]
            let aux=0
            
            function crescente(){
                for (let j = array.length - 1; j > 0; j--) {
                    for (let i = 0; i < array.length - 1; i++) {
                        if (array[i] > array[i + 1]) {
                            aux = array[i]
                            array[i] = array[i + 1]

                            array[i + 1] = aux

O que é um Fluxograma?

Um fluxograma é uma representação gráfica de um processo ou algoritmo, usado para visualizar etapas, decisões e fluxos de execução. Ele utiliza símbolos padrão para representar ações, decisões, entradas e saídas.

Elementos Básicos do Fluxograma


Início e Fim:

Representado por um oval.

Indica onde o algoritmo começa e termina.

Processos:

Representado por um retângulo.

Cada etapa ou ação do algoritmo.

Decisões:

Representado por um losango.

Pontos onde há condições ou ramificações.

Entradas/Saídas:

Representado por um paralelogramo.

Indica entrada de dados ou apresentação de resultados.

Conectores:

Setas que mostram o fluxo entre os elementos.
